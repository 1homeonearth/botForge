# botforge-platform-discord
Runtime ID: `discord`

Uses Twilight gateway/http and maps Discord protocol into BotForge typed events/intents.
No bot module receives Twilight client or Discord credentials directly.

Capabilities: discord.message.send/edit/delete, discord.embed.send, discord.thread.create, discord.channel.create/edit/delete, discord.role.create/edit/assign/remove, discord.member.timeout/kick/ban/unban, discord.auditlog.read, discord.interaction.respond, discord.command.register, discord.voice.observe, discord.webhook.manage.
